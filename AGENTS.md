# apps/mobile app dev
- use `pnpm` as package manager
- use `antd-mobile` as a ui library
- use `ahooks` as a hooks library
- use `tailwindcss` as a css framework
- use `typescript` as a language
- use `tanstack-query` as a data fetching library
- use `jest` as a testing library
